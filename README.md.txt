# 🤖 MooseBot - Chatbot para WhatsApp con OpenAI y Twilio

Este proyecto implementa un chatbot inteligente para WhatsApp usando Twilio, OpenAI y Flask. Está diseñado para automatizar respuestas, brindar información sobre retroexcavadoras Moose y mejorar la atención al cliente.

## 🚀 Características

- Conexión con WhatsApp mediante Twilio
- Motor de IA con OpenAI (GPT)
- Despliegue local con ngrok
- Automatización del primer mensaje
- Opciones de contenido para clientes: fotos, videos, ficha técnica, etc.

## 🧰 Tecnologías usadas

- Python 3
- Flask
- Twilio API
- OpenAI API
- Ngrok
- Git y GitHub

## ⚙️ Requisitos

- Cuenta de Twilio con número verificado
- API Key de OpenAI
- Ngrok instalado
- Python y pip
